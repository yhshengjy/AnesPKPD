# AnesPKPD：麻醉药理PK/PD交互式教学Notebook

## 项目简介

AnesPKPD是一套麻醉药理药代动力学/药效动力学（PK/PD）教学的交互式Google Colab Notebook资源。该资源围绕“给药方式—血浆浓度—效应室浓度—药物效应—停药恢复”的学习主线，构建了5个循序渐进的教学模块，帮助学习者通过可视化模拟理解静脉麻醉药物在体内分布、消除、效应延迟以及不同给药策略对药物作用过程的影响。

本项目适用于麻醉学本科高年级学生、住院医师规范化培训学员和研究生的PK/PD入门教学。所有Notebook均可在Google Colab中直接运行，无需本地安装Python环境。

> 说明：本项目中的模型和参数主要用于教学演示和机制理解，不用于临床麻醉给药决策或个体化用药推荐。

## 建议学习顺序

建议按照以下顺序学习：

1. **Notebook 1：一室模型与PK/PD基础**
   建立血浆浓度、效应室浓度和药物效应之间关系的基本概念。

2. **Notebook 2：二室模型与多室模型入门**
   理解麻醉药物在中央室和外周室之间的分布过程，认识多室模型相较于一室模型的意义。

3. **Notebook 3：静脉给药方式与输注策略**
   观察不同静脉给药方案如何影响浓度-时间曲线和药效变化。

4. **Notebook 4：丙泊酚PK/PD模拟**
   以丙泊酚为例，理解三室模型、效应室延迟以及输注时间对苏醒恢复的影响。

5. **Notebook 5：瑞芬太尼PK/PD模拟**
   以瑞芬太尼为例，理解快速起效、快速消退药物的PK/PD特点及其临床教学意义。

## 使用方法

### 1. 打开 Notebook

点击上表中的**Open in Colab**链接，即可在Google Colab中打开对应Notebook。


### 2. 运行Notebook

初学者请逐个运行单元格：

```text
Shift + Enter
```

**建议初学者按照Notebook中的文字说明和代码顺序逐步运行，不建议跳过前面的初始化代码单元格。**


### 3. 使用交互式控件

部分Notebook中设置了滑块、下拉菜单或交互式参数调节控件。学习者可以通过改变以下参数观察曲线变化：

* 给药剂量
* 给药间隔
* 输注速率
* 输注时间
* 药物清除率
* 效应室平衡速率常数
* EC50或Ce50
* Hill系数
* 体重等个体参数

调整参数后，图像会自动更新或需要重新运行对应代码单元格。学习者可通过比较不同参数下的浓度-时间曲线和效应曲线，加深对PK/PD概念的理解。

---

### 4. 推荐课堂使用方式

本资源可用于以下教学场景：

* 麻醉药物PK/PD基础理论课
* 研究生麻醉药理学课程
* 自学

建议教师在课堂中采用“概念讲解—参数调节—图像观察—问题讨论”的流程。例如：

1. 先讲解对应PK/PD概念；
2. 运行默认参数下的模拟结果；
3. 改变某一关键参数；
4. 引导学习者观察曲线变化；
5. 讨论曲线变化背后的药理学意义。

## 常见问题

### Q1：交互式滑块没有显示怎么办？

可能是交互控件尚未加载完成。可以尝试：

1. 重新运行包含交互控件的代码单元格；
2. 刷新Colab页面；
3. 重新打开Notebook；
4. 确认前面的依赖包安装和导入代码已经运行。


### Q2：可以修改代码和参数吗？

可以。建议先保存到个人Google Drive后再修改。学习者可根据教学需要调整模型参数、剂量方案和图像显示方式。


### Q3：这些模拟结果能否用于临床给药？

不能。本项目仅用于PK/PD教学演示。实际临床用药需结合患者情况、药品说明书、临床指南、监测指标和麻醉医师判断。
